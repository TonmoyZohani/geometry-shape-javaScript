Question 1: What is DOM? What is the purpose of DOM?

Answer: The full meaning of DOM is Document Object Model.DOM represents the HTML pages in such a way that it can be easily modified with JavaScript.
When we write a webpage in HTML, we can see it through the browser. Basically what we see in the browser and what we see in the page source are the same. But DOM represents this document in an object oriented way so that we can interact with this page with the help of DOM.

Question 2: How to select elements in DOM?

Answer: We can select elements in DOM by using various selectors. Here are,

  1. 1.getElementById()
  2. 2.getElementsByTagName()
  3. 3.getElementsByClassName()
  4. 4.querySelector()
  5. 5.querySelectorAll()

Question 3: What is Event Bubbling?

Answer: Event bubbling is a JavaScript event-handling concept.
When an element takes one event and sends that to the DOM until it reach to the parent or previous element.It is a continuous process,By using event.stopPropagation() method, we can stop event bubbling.

Question 4: What is callback function? What can we do with it?

Answer: A callback function is simply a function that is executed after another function is executed. And that's why it's called a callback function.
It helps developers to write asynchronous JavaScript code and avoids errors and problems.Because It make sure that a function won't execute before the task is finished, but will instead run immediately after the task has finished.
